# CO2 Emissions Interactive Dashboard

Interactive Shiny app exploring global CO2 emissions by country and sector.

**Data:** EDGAR GHG emissions https://edgar.jrc.ec.europa.eu/index.php
**Tools:** R (Shiny, ggplot2, dplyr)  
**Live demo:** [[link to shinyapps.io]](https://vinmk-shiny.shinyapps.io/co2_viz_0/)

## Features
- Time series visualization
- Country comparisons
- Sector breakdowns
